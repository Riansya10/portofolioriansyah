# RianDev - Premium Full-Stack Web Developer Portfolio

Selamat datang di repositori portofolio profesional saya! Website ini dirancang dengan estetika modern, performa tinggi, dan interaktivitas premium menggunakan **HTML5**, **Vanilla CSS**, dan **JS Interaktif**.

## 🚀 Fitur Utama
- **Aksen Glassmorphism & Blob Latar Belakang**: Tampilan elegan dengan efek transparansi modern dan gradasi warna dinamis yang memukau.
- **Dual-Theming (Dark & Light Mode)**: Dukungan penuh untuk mode gelap dan terang yang tersimpan secara otomatis di *localStorage* pengguna.
- **Desain Mobile-First & Responsif**: Tata letak yang dioptimalkan sepenuhnya untuk semua ukuran layar (Desktop, Tablet, dan HP) dengan penataan khusus tombol bertumpuk (*stackable CTA*).
- **Galeri Proyek Unggulan Terintegrasi**: Menampilkan proyek flagship (Kopi Keliling & ReDDA Polibatam) menggunakan mesin slider gambar (*carousel*) interaktif dengan swipe-support.
- **Integrasi Hubungi Saya (WhatsApp Direct)**: Formulir kontak interaktif yang terhubung langsung ke pesan WhatsApp pra-formulir dengan detail subjek dan isi pesan otomatis.
- **Cross-Browser SVG Collapse Fix**: Pembenahan khusus untuk mencegah ikon SVG menciut (0px) pada browser seluler (Safari iOS & Android WebViews).

## 🛠️ Tech Stack & Library
- **Struktur**: HTML5 (Elemen Semantik)
- **Gaya Visual**: Vanilla CSS (CSS Grid, Flexbox, Variable-Theming, Animasi Mikro)
- **Logika & Interaksi**: Vanilla JavaScript (ES6+, Intersection Observer untuk animasi *progress bar*, Swipe Engine)
- **Library Ikon**: Lucide Icons (Browser Build via High-Performance jsDelivr CDN)

## 💻 Cara Menjalankan Secara Lokal

1. **Clone Repositori:**
   ```bash
   git clone https://github.com/Riansya10/portofolioriansyah.git
   cd portofolioriansyah
   ```

2. **Jalankan Lokal Server:**
   - Menggunakan PHP (Direkomendasikan):
     ```bash
     php -S localhost:8000
     ```
   - Menggunakan Node.js / npx:
     ```bash
     npx http-server -p 8000
     ```
   - Menggunakan Python:
     ```bash
     python -m http.server 8000
     ```

3. **Buka di Browser:**
   Akses `http://localhost:8000` atau `http://127.0.0.1:8000` di browser kesayangan Anda.
